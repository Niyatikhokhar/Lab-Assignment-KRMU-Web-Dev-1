# Project Overview
This project demonstrates:
Responsive layout design
Use of Flexbox and CSS Grid
Clean and structured HTML
Minimalistic and modern UI
Real-world e-commerce webpage sections

# Sections Implemented
1. Navigation Bar (Sticky)
Logo, search bar, navigation links, cart icon
Stays fixed at the top using position: sticky
Designed with Flexbox
2. Hero Banner
Full-width background image
Dark overlay effect
Headline, tagline, and CTA button
3. Flash Sale Banner
Highlight bar with promotional text
Grabs user attention
4. Sidebar Filters
Category filters (Men/Women/Accessories)
Price range filters
Simple checkboxes
5. Featured Categories
Three cards: Men, Women, Accessories
Hover animation + image previews
Grid-based layout
6. Latest Products
Product cards containing image, name, price, button
CSS Grid used for neat alignment
7. Testimonials Section
Two testimonial cards
User-friendly and clean layout
8. Newsletter Subscription
Email input field + subscribe button
Encourages user engagement
9. Footer
Links + short brand text
Simple and clean design

# Main Layout Techniques Used
1. Flexbox
Used for:
Navbar
Testimonials
2. Layout responsiveness
CSS GridUsed for:
Category cards grid
Product cards grid
3. Media Queries
Used to adjust layout at different breakpoints.
4. Box-shadow, Transitions, Hover Effects
Used to enhance visual UI.
